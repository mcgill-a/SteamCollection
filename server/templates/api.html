This is an API for Steam Collection. It accepts GET requests and returns data in JSON arrays

-- Parameters --

> appid
> name
> genre
> developer
> category

You can use multiple parameters by seperating them with the '&' symbol as shown below

-- Example Usage --

> http://set09103.napier.ac.uk:9154/api/request/ -- returns data for all games
> http://set09103.napier.ac.uk:9154/api/request/?appid=730 -- returns data for game with appid '730'
> http://set09103.napier.ac.uk:9154/api/search/?developer=valve&genre=action -- returns data for games developed by Valve and include the genre Action
